# **App Name**: Amplify Advisor

## Core Features:

- Product Recommendation: AI powered product recommendations based on browsing history and customer data.
- Sales Trend Analysis: Identify trending products and predict future sales performance using machine learning models.
- Customer Segmentation: Segment customers into different categories to optimize product placement and sales strategies.
- AI-Powered Negotiation Tool: Suggest negotiation strategies based on real-time data and predicted outcomes to get the best deals.
- Interactive Dashboard: Display key metrics, sales trends, and customer insights in an intuitive user interface.
- Recommendation Feed: Display sales recommendations to the user

## Style Guidelines:

- Primary color: A vibrant blue (#2E9AFE) to evoke trust and intelligence, inspired by modern tech interfaces.
- Background color: Light gray (#F5F7FA), subtly desaturated from the primary blue, providing a clean and professional backdrop.
- Accent color: A warm orange (#FFA500), analogous to blue, to highlight key actions and calls to action.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text. Space Grotesk provides a tech-forward aesthetic for headings, while Inter ensures readability for longer text.
- Use flat, geometric icons in the primary blue and orange to represent various data points and actions.
- A clean, grid-based layout to ensure data is presented in an organized and digestible manner.
- Subtle animations for loading new data and highlighting key insights.
